# Chat-App

A web socket based chat application built with a TypeScript backend and a React + Vite frontend.

## Features

- Real-time messaging over WebSockets
- TypeScript-based backend
- React frontend
- Fast development workflow with Vite
- Toast notifications and client-side routing support

## Project Structure

```text
Chat-App/
├── backend/
└── frontend/
    └── chatapp-frontend/
```

## Tech Stack

### Backend
- TypeScript
- ws

### Frontend
- React
- TypeScript
- Vite
- React Router
- React Hot Toast
- Tailwind CSS integration

## Prerequisites

- Node.js
- npm

## Installation

Clone the repository:

```bash
git clone https://github.com/n1njaO7/Chat-App.git
cd Chat-App
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend/chatapp-frontend
npm install
npm run dev
```

## Build

### Backend

```bash
cd backend
npm run dev
```

### Frontend

```bash
cd frontend/chatapp-frontend
npm run build
```

## Notes

- The backend is started with a TypeScript build step followed by Node execution.
- The frontend is a Vite app located in `frontend/chatapp-frontend`.
